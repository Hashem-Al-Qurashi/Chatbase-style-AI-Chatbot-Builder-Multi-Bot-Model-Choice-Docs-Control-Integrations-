How to Embed the Widget

Adding our chatbot to your website is incredibly simple. Just add this single script tag to your HTML:

<script src="https://your-domain.com/widget/chatbot-widget.js" data-chatbot-slug="your-chatbot-slug" data-primary-color="#007bff" data-position="bottom-right" data-trigger-text="Chat with us" data-welcome-message="Hi! How can I help you today?" ></script>

Customization Options

The widget supports extensive customization through data attributes:

  • data-chatbot-slug: Your unique chatbot identifier (required)
  • data-primary-color: Main color theme (default: #007bff)
  • data-background-color: Background color (default: #ffffff)
  • data-text-color: Text color (default: #333333)
  • data-position: Position on page (bottom-right, bottom-left)
  • data-trigger-text: Button text (default: "Chat with us")
  • data-welcome-message: First message shown
  • data-height: Widget height (default: 500px)
  • data-width: Widget width (default: 350px)
